#  Desafio de Projeto - Relatório Financeiro

Este repositório contém o desafio de projeto do curso, focado em atualizar o relatório financeiro com foco na experiência do usuário.

##  Objetivo
Modificar o relatório criativo inicial, aplicando conceitos de:
- Posicionamento
- Contraste
- Proporção áurea
- Segmentação dos dados

O objetivo é tornar o relatório mais intuitivo e agradável para o usuário, sem perder a criatividade.

##  Estrutura do Projeto
O relatório é composto por **3 páginas**:
1. Ajustes iniciais e adequação do relatório
2. Modificação dos visuais e gráficos
3. Inserção de botões de navegabilidade e estilização
